Model Architecture

The baseline is a plain GAN: the Generator maps Gaussian noise to a patient feature vector in z-score space, and the Discriminator scores how "real" each vector looks. After training, the Generator is sampled and its outputs are inverse-transformed back to the original feature scale with discrete columns clamped to valid bounds.
